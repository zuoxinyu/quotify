# UI icons

`chevron-down.svg`, `circle-check.svg`, `circle-x.svg`, `eye.svg`, `inbox.svg`,
`info.svg`, `loader.svg`, `search.svg`, and `triangle-alert.svg` are copied from
the `gpui-component-assets` 0.5.1 bundle so `gpui-component` controls can resolve
their standard `icons/*.svg` asset paths without pulling its incompatible
optional asset crate into this application.

The bundle is published by Longbridge under Apache-2.0 and its icons are based
on Lucide icons, which are distributed under the ISC license.
